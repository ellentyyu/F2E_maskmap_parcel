# 口罩地圖

## DEMO
https://ellentyyu.github.io/F2E_maskmap_parcel/

## 功能
* 地區篩選
* 地址搜尋
* 藥局及使用者即時定位
* 購買資格及口罩庫存判別

## 技術
* OpenStreetMap 圖資
* Leaflet
* Leaflet MarkerCluster 套件
* Leaflet Locate 套件
* 響應式設計
* SCSS
* Parcel.js

## 安裝與運行
```
# parcel
npm install -g parcel-bundler

# 安裝與運行
npm install
npm run start

# 編譯(dist)
npm run build
```
